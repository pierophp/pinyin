# Pinyin Editor
A simple pinyin editor made in node.js

## Install
Run:
```
npm install
npm install -g knex
npm install -g gulp
bower install
knex migrate:latest
npm run init
```

Create a env.js file based on env.js.example



Load the Unihan Database:
```
./app/console unihan-load
```
